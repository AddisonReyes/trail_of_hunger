# Trail of Hunger

Juego 2D top-down (macroquad) donde controlas a un grupo de nomadas para cazar y agotar el ecosistema de cada nivel.

## Objetivo

- Ganas el nivel cuando no quedan animales.
- Pierdes si el hambre llega a 0.

## Controles

- Enter: avanzar / seleccionar
- Esc: pausa (en partida)
- Click izquierdo: seleccionar nomadas (click o arrastrar)
- Click derecho: ordenar mover / cazar / comer (segun el objetivo)

## Ejecutar

```bash
cargo run
```

## Notas
